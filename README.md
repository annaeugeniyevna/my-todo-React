# React ToDo

A simple and efficient task management tool built with **React**. This project was created as part of my frontend development journey to practice core React concepts.

### Key Features
* **Task Management**: Easily add and delete tasks.
* **Progress Tracking**: Real-time counter showing completed vs. total tasks.
* **Smart Cleaning**: A "Clear Completed" button that appears only when needed.
* **Persistent Data**: Uses `localStorage` to save your tasks even after refreshing the page.

### Tech Stack
* **React** (Vite)
* **Hooks**: `useState`, `useEffect`
* **JavaScript** (ES6+)
* **CSS3**
